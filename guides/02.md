# ツールについて知ろう

Webアプリケーションを作るには、いくつかのツールが必要です。このページでは、これから使うツールについて説明します。

## 使用するツール

### 1. テキストエディタ - Visual Studio Code

**テキストエディタ**は、プログラムのコードを書くためのツールです。

今回は **Visual Studio Code（VS Code）** を使います。
VS Codeは、

- 無料で使える
- 設定なしですぐに使える
- 非常に完成度が高い
- 多くのプログラマーに使われている

他にも様々なテキストエディタがありますが、初心者の方にはVS Codeをおすすめします。

**ダウンロード** [https://code.visualstudio.com/](https://code.visualstudio.com/)

### 2. ターミナル（コマンドライン）

**ターミナル**は、コンピュータに命令を文字で伝えるツールです。

マウスでクリックする代わりに、キーボードで命令を入力します。最初は慣れないかもしれませんが、プログラミングには欠かせないツールです。

#### ターミナルの開き方

**macOSの場合**
1. Finderを開く
2. 「アプリケーション」→「ユーティリティ」→「ターミナル」をクリック

または、Spotlightで「ターミナル」と検索してください。

**Windowsの場合**
1. スタートメニューを開く
2. 「Windows PowerShell」または「コマンドプロンプト」を検索
3. クリックして開く

#### 重要な注意点

⚠️ **macOS用の手順はWindowsでは動きません**

OSによってコマンドが異なる場合があります。このガイドでは、各OSで動くコマンドを記載していますので、ご自身の環境に合わせて実行してください。

### 3. Webブラウザ

作成したアプリケーションを表示して確認するために使います。

以下のいずれかのブラウザを使用してください。

- Google Chrome（推奨）
- Firefox
- Safari
- Microsoft Edge

**推奨** Google Chrome または Firefox
これらのブラウザには、開発者向けの便利な機能が充実しています。

### 4. その他のツール

アプリ開発には、以下のソフトウェアも必要です。

- **PHP** - Laravelを動かすためのプログラミング言語
- **Composer** - PHPのパッケージ管理ツール
- **Node.js** - JavaScriptの実行環境
- **Laravel** - Webアプリケーションフレームワーク

これらは次の「インストール・レシピ」でインストールします。

## まとめ

今回使用するツールは

| ツール | 用途 |
|--------|------|
| Visual Studio Code | コードを書く |
| ターミナル | コマンドを実行する |
| Webブラウザ | アプリを表示する |
| PHP / Composer / Node.js | アプリを動かす |

次は「インストール・レシピ」で、これらのツールをインストールしましょう！

---

**ヒント**
- ターミナルで間違ったコマンドを入力しても、パソコンが壊れることはありません
- わからないことがあったら、コーチに質問しましょう
- 最初はターミナルに慣れないかもしれませんが、使っているうちに慣れてきます
