# インストール・レシピ

アプリ作りの準備として、必要なソフトウェアをインストールしましょう。

## 作業時間の目安

- **実際の作業時間** 約5分
- **インストール待ち時間** 15〜30分

インストール中は他のことをしていても大丈夫です。

---

## 重要なお知らせ

インストールの手順については、**あまり細かく理解しようとしなくて大丈夫**です。

今は「アプリを作る準備」と考えてください。使っていくうちに、それぞれのツールの役割が分かってきます。

⚠️ もしエラーが出たら、**すぐにメンターに相談してください**

## macOS

### 1. Homebrewのインストール

Homebrewは、macOS用のパッケージ管理ツールです。

ターミナルを開いて、以下のコマンドを実行してください：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

パスワードを求められたら、macOSのログインパスワードを入力してください（画面には表示されませんが、入力できています）。

### 2. PHPのインストール

```bash
brew install php
```

インストールが完了したら、バージョンを確認しましょう。

```bash
php --version
```

`PHP 8.2` 以上が表示されればOKです。

### 3. Composerのインストール

Composerは、PHPのパッケージ管理ツールです。

```bash
brew install composer
```

確認

```bash
composer --version
```

### 4. Node.jsのインストール

```bash
brew install node
```

確認

```bash
node --version
npm --version
```

両方のコマンドでバージョン番号が表示されればOKです。

### 5. Visual Studio Codeのインストール

1. [https://code.visualstudio.com/](https://code.visualstudio.com/) にアクセス
2. 「Download for Mac」をクリック
3. ダウンロードしたファイルを開いて、アプリケーションフォルダにドラッグ

### インストール完了！

すべてのツールがインストールできたら、次のステップに進みましょう。

## Windows

### 1. PHPのインストール

#### 1-1. XAMPP（ザンプ）をインストール

XAMPPは、PHPを簡単に使えるようにするパッケージです。

1. [https://www.apachefriends.org/](https://www.apachefriends.org/) にアクセス
2. 「Windows向けXAMPP」をダウンロード
3. ダウンロードしたファイルを実行してインストール
4. インストール先は `C:\xampp` のまま（変更しない）

#### 1-2. 環境変数の設定

PHPをどこからでも使えるようにします。

1. スタートメニューで「環境変数」と検索
2. 「システム環境変数の編集」をクリック
3. 「環境変数」ボタンをクリック
4. 「システム環境変数」の「Path」を選んで「編集」
5. 「新規」をクリックして `C:\xampp\php` を追加
6. 「OK」で閉じる

**確認** PowerShellを**新しく開いて**以下を実行

```powershell
php --version
```

`PHP 8.2` 以上が表示されればOKです。

### 2. Composerのインストール

1. [https://getcomposer.org/download/](https://getcomposer.org/download/) にアクセス
2. 「Composer-Setup.exe」をダウンロード
3. インストーラーを実行
4. すべてデフォルト設定のまま進める

**確認** PowerShellを**新しく開いて**

```powershell
composer --version
```

### 3. Node.jsのインストール

1. [https://nodejs.org/](https://nodejs.org/) にアクセス
2. 「LTS」版（推奨版）をダウンロード
3. インストーラーを実行
4. すべてデフォルト設定のまま進める

**確認** PowerShellを**新しく開いて**

```powershell
node --version
npm --version
```

### 4. Visual Studio Codeのインストール

1. [https://code.visualstudio.com/](https://code.visualstudio.com/) にアクセス
2. 「Download for Windows」をクリック
3. ダウンロードしたファイルを実行してインストール

### インストール完了！

すべてのツールがインストールできたら、次のステップに進みましょう。

## よくあるエラー

### 「コマンドが見つかりません」と表示される

**原因** ターミナル（PowerShell）を閉じていない

**解決方法** ターミナルを**完全に閉じて、新しく開き直してください**

環境変数の変更は、新しく開いたターミナルでのみ反映されます。

### Homebrewのインストールが失敗する（macOS）

**解決方法** メンターに相談してください

Xcodeのコマンドラインツールのインストールが必要な場合があります。

### どうしてもうまくいかない場合

**重要** 遠慮なくメンターに相談してください！

環境構築は初心者にとって最も難しい部分の一つです。プロのエンジニアでも、環境構築で時間がかかることがあります。
