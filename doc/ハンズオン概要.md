# ハンズオン概要

## 対象者
- プログラミング未経験者
- プログラミング初学者
- 学生

## 目的
実際に手を動かしてプログラミングを経験し、動きのあるアプリケーションを成果物として開発する経験を実感していただきます。

## 実施要件
- **フレームワーク**: Laravel 12
- **開発環境**: macOS または Windows
- **エディタ**: Visual Studio Code
- **題材**: 日記アプリケーション
- **所要時間**: 半日程度（最大6時間を想定）

## 実装内容
- SQLiteによるデータベース構築
- 日付選択機能
- 日記の登録・更新・削除機能
- ログイン機能は実装しない（余力があれば対応）

## 技術選定
- **フロントエンド**: Bladeテンプレート
  - 初学者にとって学習曲線が緩やか
  - HTML/CSS/PHPの基本知識で理解可能
  - リクエスト/レスポンスの流れが明確でデバッグが容易
  - Laravelの基礎を学ぶ土台として最適
- **スタイリング**: Tailwind CSS（またはdaisyUI）
  - 見栄えのある成果物で達成感を高める

## 補足事項
- JavaScriptは最小限に抑え、フォーム送信による従来のMVC構造を学ぶ

## 画面構成
2画面構成（シンプルなCRUD）

### 一覧画面
- 日記一覧表示
- 月単位のプルダウンで日付絞り込み
- 新規作成ボタン
- 各日記をクリックで編集画面へ遷移

### 編集画面
- 日付入力（HTML5 date input）
- 内容入力（テキストエリア）
- 保存・削除・戻るボタン
